 Sistema de Controle de Contratos — Flask + SQLite
Um sistema simples para gerenciamento de contratos em Flask + SQLite. Ideal para estudos e portfólios de iniciantes.


Um sistema simples de gerenciamento de contratos desenvolvido em Python (Flask), ideal para iniciantes que desejam aprender sobre:

CRUD (Create, Read, Update, Delete)
Banco de dados SQLite
Templates HTML com Jinja2
Estilização com CSS puro
Lógica de alertas automáticos de vencimento

 Funcionalidades Principais

Cadastrar contratos com cliente, valor e data de vencimento
Editar e excluir contratos já criados
Status automático (Ativo / Vencido) com base na data atual 
Alertas automáticos de vencimentos próximos (até 7 dias)
Interface simples e moderna com HTML e CSS puro
Banco SQLite, leve e sem dependências externas

 Tecnologias Utilizadas
Tecnologia	Descrição
Python 3	Linguagem principal
Flask	Framework web minimalista
SQLite3	Banco de dados embutido
HTML + CSS	Interface do usuário
Jinja2	Motor de templates integrado ao Flask
